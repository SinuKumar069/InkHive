# API Reference

## HTTP endpoints

### POST /api/trigger-inngest

Starts content generation for a project.

```json
{
  "projectId": "proj_abc123",
  "inputType": "topic",
  "inputContent": "Single idea to expand"
}
```

### POST /api/upload

Accepts a file upload and returns a URL.

```json
{
  "url": "https://example.com/file.png"
}
```
