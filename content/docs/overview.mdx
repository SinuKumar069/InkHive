# Overview

InkHive turns a single idea into a full content kit: long-form blog, SEO metadata, social posts, and email copy. The workflow is designed for fast iteration, real-time progress, and reliable publishing.

## Flow at a glance

1. Create a content project from a topic or seed article.
2. Generation runs in the background, with progress updates in the dashboard.
3. Drafts are editable in the UI and saved to Convex in real time.
4. Publishing fans out to multiple platforms with per-channel status updates.

## Core building blocks

- Next.js app router UI with Clerk auth.
- Convex for realtime data, queries, and mutations.
- Inngest for orchestration and retries.
- Google Generative AI for content generation.
