# Deployment

## Environment variables

- `NEXT_PUBLIC_CONVEX_URL`
- `CLERK_JWT_ISSUER_DOMAIN`
- `GOOGLE_API_KEY`
- `RESEND_API_KEY`
- `INNGEST_API_KEY`

## Checklist

1. Set Convex deployment URL.
2. Configure Clerk JWT issuer.
3. Add AI and email provider keys.
4. Configure Inngest credentials.
5. Verify publish platform credentials.
